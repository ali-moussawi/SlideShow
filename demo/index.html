Installation

You can download files from Github or install through NPM:

npm install simpleslider-js

or If you prefer files from CDN:

https://unpkg.com/simpleslider-js@VER/dist/simpleSlider.min.js

https://unpkg.com/simpleslider-js@VER/dist/simpleSlider.min.css

*Remember to replace the VER with the script version.
